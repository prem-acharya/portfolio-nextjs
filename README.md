# Prem Acharya Portfolio

A modern portfolio website built with **Next.js**, **Tailwind CSS**, and **TypeScript**.

## Features

- **Responsive Design:** Mobile, tablet and desktop/laptop responsive.
- **Type Safety:** Developed with TypeScript.
- **Modern UI:** Clean interface showcasing projects and experience.
- **Theme Switch:** Toggle between light and dark mode with a smooth animation

## Getting Started

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/prem-acharya/portfolio-nextjs.git
cd portfolio-nextjs
npm install
```

### Running Locally

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.


## Deployment

Deploy your portfolio website using platforms like [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/).
